# MyExpenses Assets

This directory contains application assets including icons and splash screens.

## Required Assets

### Mobile App (Expo)

- **icon.png**: 1024x1024px - App icon
- **splash.png**: 1284x2778px - Splash screen
- **adaptive-icon.png**: 1024x1024px - Android adaptive icon
- **favicon.png**: 48x48px - Web favicon

### Desktop App (Electron)

- **icon.png**: 512x512px - App icon (cross-platform)
- **icon.ico**: Windows icon
- **icon.icns**: macOS icon

## Creating Icons

You can create your own icons or use the provided placeholders. For production apps, it's recommended to use professional icon design.

### Tools for Icon Creation

- **Online**: favicon.io, canva.com
- **Desktop**: Adobe Illustrator, Figma, Sketch
- **Icon Converters**: 
  - PNG to ICO: convertio.co
  - PNG to ICNS: cloudconvert.com
