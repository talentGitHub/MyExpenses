# MyExpenses

Cross-platform expense tracking application for Windows, Android, and iOS.

## Getting Started

### Installation

```bash
npm install
```

### Windows Users

If you encounter PowerShell execution policy errors when running npm commands on Windows, please refer to [TROUBLESHOOTING_WINDOWS.md](./TROUBLESHOOTING_WINDOWS.md) for solutions.

## Development

```bash
npm test
```

## CI/CD

This project uses GitHub Actions for continuous integration. The workflow is configured to handle Windows PowerShell execution policy issues automatically.
